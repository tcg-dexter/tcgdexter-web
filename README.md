# Dexter's Collection — tcgdexter.com

Pokémon TCG card shop and community hub. Built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

Push to GitHub, then import the repo at [vercel.com/new](https://vercel.com/new). Zero config needed — Vercel auto-detects Next.js.

## Tech stack

- **Next.js 14** — App Router
- **TypeScript**
- **Tailwind CSS** — custom dark theme with TCG energy accent colors
- **Geist** — variable font from Vercel
